# Instalação no PC (Windows)

## 1) Pré-requisitos
- Python 3.11 ou superior
- Git (opcional, se você baixar zip)

## 2) Instalar
1. Extraia o projeto em uma pasta.
2. Execute `instalar_pc.bat`.

## 3) Iniciar o dashboard
- Execute `iniciar_dashboard.bat`.

## 4) Atualização
- Baixe a versão mais recente e substitua os arquivos do projeto.
- Rode novamente `instalar_pc.bat` para garantir dependências atualizadas.
